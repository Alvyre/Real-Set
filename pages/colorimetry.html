Why Colorimetry ?

When we take a photo of our scene, we have to control all the parameters about the image. Colorimetry is one of these parameters.
The colors are important to make a good comparison between the real model and the 3D model.

X-Rite Colorchecker Passport

X-rite is one of the world leaders about sciences and color management technologies. X-rite provide services on a large scale, from medicine, printing, to photography or even automobile sector. X-rite has a large range of products which is very adaptive.
We choose the ColorChecker Passport for this project. The chart is relatively complete and can be taken or placed wherever we want.

The chart is composed of three panels :

  • Creative Enhancement Target

    - Evaluate and optimize shadow details or highlight clipping with gray ramp patches.
    - Use as visual reference for color spectrum and color adjustment with HSL (hue, saturation, lightness) patches.
    - Warm or cool images.

  • Classic Target

    - Industry standard color reference target for creating DNG profiles and for evaluating specific colors

  • White Balance Target

    - Create custom in-camera white balance for a consistent white point in a set of images, eliminating the need to correct each image later

How to use my ColorChecker Passport ?

A. Create my color profile :

Requirements

- DSLR (Digital Camera) using the RAW format
- Adobe Photoshop with CameraRAW or Adobe Lightroom
- ColorChecker Software + Lightroom plugin (if Lightroom) : (Download)

  1. Calibrate the white balance on your DSLR by placing the White Balance Target in the scene. The target must be illuminated by the main light source.
  2. Now, replace the White Balance target by the Classic one.
  3. Take a RAW photo of the target. You have to be the most perpendicular as possible in order to make the detection of the target easier on the software.
  4. If your DSLR uses .DNG as RAW format, go to next step, else open your RAW photo in Lightroom or CameraRAW and export in .DNG

    5. Using Adobe Lightroom :

  5. Import your photo .DNG of the ColorChecker, crop on the target and export it as color profile (right clic > export > ColorChecker Passport).
  6. Close and restart Lightroom.

    Using ColorChecker Passport software :

  1. Import your photo .DNG in the ColorChecker Passport software. Your target will be automatically detected. Else, follow the instructions to select the active zone.
  2. Click on "Create profile" and save the file .dcp.
  3. Move your file .dcp to C:\Users\[USER_NAME]\AppData\Roaming\Adobe\CameraRaw\CameraProfiles.

B. Apply a color profile :

Requirements

- DSLR (Digital Camera) using the RAW format
- Adobe Photoshop with CameraRAW or Adobe Lightroom
- ColorChecker Software + Lightroom plugin (if Lightroom) : (Download)
- DNG profile manager software (Download)
- Color profile loaded (see A.)

Using Adobe Lightroom :

  1. Import your RAW photo in Lightroom.
  2. Select the photo and click on "Develop".
  3. Choose your profile in the section "Camera Calibration" > "Profile".

Using Adobe CameraRAW :

  1. Import your RAW photo into Photoshop, it will be automatically opened with CameraRAW.
  2. Click on "Camera Calibration" (with the camera icon).
  3. Choose your profile in the menu "Camera Profile".
  4. Now you're able to :
    - Use the photo in photoshop by clicking on "Open Image" on the bottom right corner.
    - Save the photo by clicking on "Save the image" on the bottom left corner.

C. Manage the color profiles :

Requirements

- DNG profile manager software (Download)

DNG profile manager, is a software, provided by X-rite, giving access to manage easily your colors profiles. Several options are available to sort the profiles (by camera for example). You can also activate/desactivate profiles.

  1. Activate/desactivate a color profile :
    1. Execute DNG Profile Manager.
    2. Active or desactivate a profile by ticking the case related.
  2. Delete a color profile :
    1. To delete profiles, select the profiles by ticking the cases related.
    2. "File" > "Delete", then confirm.

Shortcuts :

  • You can delete a profile with CTRL + D.
  • To refresh your color profiles list, press CTRL + L or select the option "Reload Profiles" in "File" menu.
  • You can manually manage your profiles (adding/deleting) via the finder of your OS by pressing CTRL + F or by selecting "Open the folder of DNG profiles" in "File" menu.

    • Warning Deleting a profile in DNG Profile Manager is definitive.
