What is an IES Profile ?

The IES (Illumation Engineering Society) format is an easy way to represent light profiles. Theses profiles are generally used to render interior architecture. They can be used also to define an atmosphere and emotions for a location (bar, exhibition with soft lights, ..). Theses profiles can show the physical repartition of the light from whichever source of light.

In Renderman, IES profiles can be applied to a Geometric Area Light to create a sophisticated light and then to reproduce a real light.

This collection of Geometric Areas Lights (cf image) and the associated profiles are ready-to-use in a Maya scene.

We chose these profiles to have a good representation of the light in our model. In this page you will find answers to these questions :

  • Where can I find IES profiles ? How to build them ?
  • How to visualize a profile ?

Preview an IES Profile

Requirements

- IESViewer (Download)

Previewing an IES profile before to use it in Maya is possible with the software "IESViewer". It's a photometric viewer which is able to preview, find and convert photometric files (IES, LDT).

IESViewer functions :
  • Photometric viewing
  • Photometric file details
  • Real time rendering of the photometric
  • Comparison between two profiles.
  • Converting from LDT to IES format

Create a profile

Requirements

- Creator IES (Download)

You can make your own IES profile by using the software Creator IES.
With Creator IES, you are able to make IES profiles templates differents from most known and used profiles.
Creator IES can create a profile from scratch or to modify and customize an existing one. The window shows 2 representations : Curbe Editor and Picture thumbnail

The Curve Editor is a graph of the IES light template (red line of the photo). The length of the curve shows the differents parts of the templace which are easily modified by clicking&dropping wherever on the curve.

We can also set pamameters on the light :

  • Edition zone
  • Luminosity
  • Distance from the wall
On the right side, we can see the resulting image of the curve, created in the editor in real time.
